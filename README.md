GitHub Actions CI Pipeline Lab
Учебный DevOps-проект, демонстрирующий настройку CI pipeline с использованием GitHub Actions.
Pipeline triggers
Pipeline запускается в трёх случаях:
push в ветку main
создание Pull Request в ветку main
ручной запуск (workflow_dispatch)
Workflow
Pipeline выполняет следующие шаги:
Checkout репозитория
Запуск тестового скрипта
